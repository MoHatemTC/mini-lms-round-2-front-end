# Lessons Feature

Lesson management within courses. Supports video lessons, text content, ordering, and lesson-level progress tracking.

## Structure

- **pages/** - Page-level components for this feature
- **components/** - Feature-specific UI components
- **hooks/** - Custom React hooks for this feature
- **services/** - API service calls
- **store/** - Redux Toolkit state management (slice, selectors, thunks)
- **utils/** - Feature-specific utility functions
- **constants/** - Feature-specific constants
- **validation/** - Zod validation schemas
